# Changelog

All notable user-visible changes are documented here. The project follows
[Semantic Versioning](https://semver.org/) while it develops toward a stable `1.0.0` interface.

## [Unreleased]

### Added

- Snell v6 server support for compatible sing-box 1.14 Preview cores.
- Stable and Preview core release channels with trust-bound artifact acquisition.
- Repository governance, security scanning, dependency updates, and tag-driven GitHub Releases.

### Changed

- The project now identifies as an independently developed Python application derived from the
  original shell manager.

### Security

- Release artifacts include SHA-256 checksums and GitHub build-provenance attestations.
